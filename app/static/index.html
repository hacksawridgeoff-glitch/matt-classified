Исходный код Как проверить безопасность

Почему это безопасно?

Информация на сервер передаётся только в зашифрованном виде, без ключей расшифровки. Ключ хранится внутри вашей ссылки после символа #. Его видят только вы и код этой страницы — на сервер он не попадает.

Как убедиться в этом самостоятельно:

  1. Нажмите F12 в браузере (или Cmd+Opt+I на Mac) — откроются инструменты разработчика.
  2. Перейдите во вкладку Network.
  3. Создайте записку.
  4. В списке запросов найдите POST /api/notes. Кликните по нему и откройте вкладку Payload или Request.
  5. Вы увидите, что на сервер уходит только поле ciphertext — бессмысленный набор символов, и iv. Текста вашей записки и ключа там нет.
  6. Посмотрите на URL готовой ссылки: всё, что после #, — это ключ. По стандарту HTTP эта часть никогда не отправляется на сервер.
